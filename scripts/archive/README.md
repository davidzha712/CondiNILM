# Archived Scripts

These scripts were used during development and paper preparation. They are
**not** part of the standard training/evaluation workflow and are kept here
for reference only.

| Directory      | Purpose                                                        |
|----------------|----------------------------------------------------------------|
| `tensorboard/` | TensorBoard log cataloguing, metric export, and best-run queries |
| `paper/`       | Result extraction, figure generation, and table formatting for the paper |
| `profiling/`   | VRAM estimation and preprocessing visualisation utilities       |
